# Tic Tac Toe

A classic Tic Tac Toe game built using purely HTML, CSS, and JavaScript. It features an aesthetic UI and allows players to choose from three different play modes.

## Features

- **2 Players Mode**: Play locally against another human.
- **Vs Easy AI**: Play against an AI that makes random moves. Great for beginners!
- **Vs Hard AI**: Play against an unbeatable AI powered by the Minimax algorithm.
- **Dynamic UI**: Highlights the winning combination and displays a clear turn indicator.
- **Responsive Design**: Clean and minimalistic game board.

## Technologies Used

- **HTML5:** Structures the game board and UI elements.
- **CSS3:** Provides modern styling, including colors, typography, and visual feedbacks for turns and wins.
- **JavaScript (Vanilla):** Implements the game logic, UI updates, and the AI opponents (including the Minimax algorithm).

## How to Run

1. Simply clone or download the repository to your local machine.
2. Open the `index.htm` file in any modern web browser.
3. Select your desired play mode from the dropdown menu and start playing!

## How to Play

- The game starts with Player `X`.
- Click on any empty square to place your mark.
- If you're playing against the AI, it will automatically make its move as `O`.
- The first player to align 3 of their marks horizontally, vertically, or diagonally wins.
- If all squares are filled and no player has 3 in a row, the game ends in a draw.
